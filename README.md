# Ecommerce
This is a simple ecommerce project made in react fetching a fake store API.

# Tech

- React
- Context API
- Styled Components

# Features
- Show categories
- Show products from a categorie
- Show product detail
- Add product to the cart
- Remove product from the cart
- Show cart
- Clear cart
